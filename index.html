Automated CAPTCHA Generation from Annotated Images using Encoder Decoder

Google recently came up with a system capable of breaking current OCR CAPTCHAs. This necessitates the need for more complex CAPTCHAs which cannot be easily compromised. Also, currently employed CAPTCHA systems are static in nature which mandates the need for a dynamic CAPTCHA generation mechansim.

The proposed system dynamically generates multiple choice questions based on the image content which serve as the CAPTCHA. The system follows a GRU RNN (Gated Recurrent Unit Recurrent Neural Network) encoder-decoder architecture where the image is sent to the encoder and decoder generates the question. In this project, I actively participated in the idea conception and the system design phase. I also involved myself in the dataset generation for model training and developed the GRU RNN encoder-decoder backend infrastructure for question generation.

This project has been accepted for publication at the 2016 IEEE 15th ICIT Conference Proceedings.

Automated Content Suggestion for Document Writing

The process of document writing can tend to be a rather laborious one as it entails the writer to perform a lot of research on the topic at hand. The writer will need to go through the hassle of performing multiple Google searches, going to various links, understanding the information on each weblink and then finally putting together a coherent document.

This project speeds up the document writing process by dynamically providing web based content (both textual and media) suggestions depending on the current document context. The suggestions are provided in a tile format on one side of the editor, thereby, enabling seamless readability. The user can also readily accommodate the suggestions into the document.

This project has been accepted for publication at the 2016 IEEE ICCIC Conference Proceedings.

Cricket Match Summary Generation

In this project, we developed a system capable of generating match summaries given the match commentaries. The commentaries were scraped from ESPN CricInfo and the commentaries dataset was created. Named Entity Recognition was performed on each commentary using a Vanilla Recurrent Neural Network (RNN). Different match events like sixes, fours, wickets etc. were tagged using a MaxEnt model. A Natural Language Generation (NLG) module then generated a match summary based on the outputs of the RNN and MaxEnt Model. I actively contributed to the dataset generation, event tagging and the NLG modules.

Duration: Dec 2015


Cricket Match Score Prediction

The aim of the project was to predict match scores between two Indian Premier League (IPL) teams based on previous IPL match statistics. Player vs player statistics and individual player statistics were scraped from ESPN CricInfo. The players were divided into clusters using Apache Mahout's K-means clustering algorithm. Using Apache Spark, a map reduce algorithm was written to calculate probabilities of occurrence of different events like 1s scored, 4s scored, wicket etc. Finally, given 2 IPL team rosters, a ball by ball simulation was done and a prediction algorithm was used to determine the final score for both the teams and declare the winner. A random process was initiated to select the events and determine the score based on the probabilities calculated previously. In this project, I worked on the Mahout clustering module and the Spark map reduce algorithm for computing event probabilities.
